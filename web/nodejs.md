  ```bash
  npm install learnyounode -g
  ```

  - step01：你好，世界  
    - console.log  
  - step02：婴儿学步  
    - process.argv 访问命令行参数

  - step03：第一个 I/O  
    - fs.readFileSync 同步读取文件

    - fs.readFile 异步读取文件

  - step11：文件服务器

    - http 对象的使用
  - step12：大写转换器  
  - step13：HTTP JSON API 服务器
